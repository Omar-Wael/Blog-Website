# Blog Website

This is a simple website that allows users to create, view, edit, or delete posts.

## Installation

1. Clone the repository.
2. Install the required dependencies using `composer install`.
3. Start the server using `php artisan serve`.

## Usage

To create a new post:

-   Click on the "Create Post" button and fill in the details.

To view a post:

-   Click on the post title to view the full post.

To edit or delete a post:

-   Click on the "Edit" or "Delete" button next to the post.
